## 任务 1：固定值改为响应式

### 1.1 首页 (index.vue)

**当前固定值**：
```css
.header {
  margin-top: 80rpx;      /* → clamp(40rpx, 5vh, 100rpx) */
  margin-bottom: 100rpx;   /* → clamp(60rpx, 8vh, 140rpx) */
}

.entry-icon {
  width: 144rpx;           /* → clamp(80rpx, 20vw, 160rpx) */
  height: 144rpx;          /* → clamp(80rpx, 20vw, 160rpx) */
}
```

**优化目标**：
- 使用 `clamp(min, preferred, max)` 实现平滑响应
- 高度相关使用 `vh` 单位
- 宽度相关使用 `vw` 单位

### 1.2 我的页 (wode.vue)

```css

.stat-value {
```

## 任务 2：图片懒加载和格式优化

### 2.1 图片组件优化

**优化前**：
```vue
<image class="entry-icon" src="/static/icons/xiangyun.png" />
```

**优化后**：
```vue
<image 
  class="entry-icon" 
  src="/static/icons/xiangyun.png"
  mode="widthFix"
  lazy-load
  show-menu-by-longpress
/>
```

### 2.2 图片模式选择

| 场景 | 推荐 mode | 说明 |
|------|----------|------|
| 图标/Logo | `aspectFit` | 完整显示 |
| 商品图 | `aspectFill` | 填充裁切 |
| 头像 | `widthFix` | 宽度固定 |
| 背景图 | `scaleToFill` | 拉伸填充 |

### 2.3 占位图方案

```vue
<image 
  :src="item.image"
  mode="aspectFill"
  :lazy-load="true"
  @error="onImageError"
  @load="onImageLoad"
/>
```

### 3.2 主题切换逻辑

**位置**：App.vue

```javascript
// 主题状态
const theme = ref('light')

// 初始化主题
onLaunch() {
  const savedTheme = uni.getStorageSync('theme') || 'light'
  this.applyTheme(savedTheme)
}

// 切换主题
applyTheme(theme) {
  this.theme = theme
  uni.setStorageSync('theme', theme)
  // 通知所有页面更新
}

// 监听系统主题变化
onShow() {
  uni.onThemeChange((res) => {
    if (uni.getStorageSync('theme') === 'auto') {
      this.applyTheme(res.theme)
    }
  })
}
```
