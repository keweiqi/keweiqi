## 添加本地经文

### 步骤
1. 打开 `static/data/articles.json` 文件
2. 按照以下格式添加新经文：

```json
{
  "道德真经": "道可道，非常道。名可名，非常名。...",
  "你的新经书名": "这里放经文内容"
}
```

3. 保存文件，重新编译应用即可

## 配置阿里云云端

### 步骤1：获取阿里云访问地址
1. 登录你的阿里云后台
2. 找到云存储/对象存储服务
3. 上传你的经文文件
4. 复制文件的访问地址

### 步骤2：修改配置文件
打开 `config/yunpeizhi.js`，修改以下内容：

```javascript
const peizhi = {
  // 开启云端功能：把 false 改成 true
  qiyongyunduan: true,

  ali: {
    // 道藏目录地址（可选）
    daojing_list: 'https://你的阿里云地址/daojing_list.json',

    // 大藏经目录地址（可选）
    fojing_list: 'https://你的阿里云地址/fojing_list.json',

    // 经文内容地址前缀（重要！）
    jingwen_qianzhui: 'https://你的阿里云地址/articles/'
  },

  chaoshi: 5000
}
```

## 云端上传文件格式

### 1. 经文内容文件
在云存储创建一个 `articles` 文件夹，里面放每本经书的 JSON 文件：

**文件名规则**：把书名里的特殊字符换成下划线，加 `.json` 后缀

例如：
- `道德真经.json`
- `般若波罗蜜多心经.json`

**文件内容格式**：
```json
{
  "content": "这里放完整的经文内容"
}
```

或者直接放字符串：
```json
"这里放完整的经文内容"
```

### 2. 目录文件（可选）
如果需要从云端更新目录，可以上传：

**daojing_list.json**
```json
[
  { "name": "道德经", "category": "真经典籍" },
  { "name": "太上老君说常清静经", "category": "真经典籍" }
]
```

**fojing_list.json**
```json
[
  { "name": "般若波罗蜜多心经", "category": "大乘经典" },
  { "name": "金刚经", "category": "大乘经典" }
]
```

## 常见问题

### Q: 下载按钮点击没反应？
A: 检查是否配置了云端地址，并且 `qiyongyunduan` 设置为 `true`

### Q: 下载后离线不能看？
A: 下载成功后会自动保存到本地，离线也能看的

### Q: 怎么删除已下载的经文？
A: 点击"删除"按钮，确认后就删除了
